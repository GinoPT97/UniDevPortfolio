Aprire il file cron da terminale con questa riga di comando :

crontab-e

Righe da inserire :

0 * * * * source ~/.bashrc && completo

#Se non inserisco source ~/.bashrc non carica gli alias e quindi non li riconosce


Per concedere i permessi aprire visudo :

sudo visudo

Aggiungere queste righe sotto alla riga # User privilege specification :

kenobi ALL=(ALL) NOPASSWD: /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/Aggiornamenti.sh

sudo chmod u+x InstallApp.sh

sudo chmod u+x PermessiSSh.sh

Permessi

# Modifica le autorizzazioni sul file chiave SSH privata
sudo chmod 600 ~/.ssh/id_rsa

# Aggiungi la chiave SSH privata all'agente SSH
ssh-add ~/.ssh/id_rsa

# Verifica che la chiave SSH privata sia stata aggiunta all'agente SSH
ssh-add -l
