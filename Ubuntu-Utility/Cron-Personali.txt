Aprire il file cron da terminale con questa riga di comando : 

crontab-e

Righe da inserire :

0 * * * * sudo -u kenobi /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/Aggiornamenti.sh
30 * * * * sudo -u kenobi /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/Ottimizzazioni.sh
0 */1 * * * clamscan -r -i / --remove | head -c 1G > /home/kenobi/Documenti/scan_result.txt

Per concedere i permessi aprire visudo :

sudo visudo

Aggiungere queste righe sotto alla riga # User privilege specification :

kenobi ALL=(ALL) NOPASSWD: /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/Aggiornamenti.sh
kenobi ALL=(ALL) NOPASSWD: /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/Ottimizzazioni.sh
kenobi ALL=(ALL) NOPASSWD: /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/InstallApp.sh

Concedere permessi ed eseguibili :

sudo chmod u+x InstallApp.sh
sudo chmod u+x Aggiornamenti.sh
sudo chmod u+x Ottimizzazioni.sh
