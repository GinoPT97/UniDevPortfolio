Analisi dello script

Traccia

La traccia richiedeva che:

  • Data una cartella
  • Una estensione scelta

Venisse eliminata ricorsivamente tutti i file che contenessero quella estensione.

Opzioni abilitate

set -e # Uscita immediata nel caso in cui un comando dia un exit status diverso da 0

shopt -s globstar dotglob nullglob

Per maggiori informazioni (dal manuale gnu):

  • set builtin;
  • shopt builtin.

Scelte implementative

Per una maggiore brevità di scrittura, faccio uso delle seguenti funzioni:

  1. print_ok(), print_info() e print_error();
    • Per una stampa su stout con colori diversi in base all'informazione da comunicare;
  2. usage(), per informare l'utente sull'utilizzo corretto dello script;
  3. searchNdelete(), per eseguire le operazioni di ricerca ed eliminazione dei file.

Nel dettaglio - usage

usage() {
  print_info "Questo script elimina file che matchano una estensione (data dall'utente) dalla directory corrente ricorsivamente.\n"
  print_ok "\t\tUtilizzo: ./CleanDir.sh estensione-file"
  exit 1
}

Nel dettaglio - funzioni ausiliare

print_ok() {
    printf "\e[32m%b\e[0m" "$1"
    printf "\n"
}

print_info() {
    printf "\e[36m%b\e[0m" "$1"
    printf "\n"
}

print_error() {
    printf "\e[31m%b\e[0m" "$1"
    printf "\n"
}

Nel dettaglio - searchNdelete

searchNdelete(){
  print_info "Controllo la presenza di file che matchano la richiesta"
  for f in **/*; do
    printf "%s\n" "$f" | grep $estensione >>  "$TmpStack"
  done 

  if [ -s "$TmpStack" ]; then
    echo " "
    else
    print_info "Nessun file trovato che matcha l'estensione richiesta!\n"
    exit 1
  fi

  sed -ie 's|^|./|g' "$TmpStack"  # Aggiungiamo ./ all'inizio di ogni riga

  print_error "Sto per eliminare i seguenti file che matchano la richiesta:\n"

  while read -r elem; do
    printf '%s\n' "$elem"
  done < "$TmpStack"


  print_error "\n\nSei sicuro? (Si/No)"

  read -r p

  if [[ "$p" != [Ss] ]]; then
    print_error "Uscita..."
    exit 1
  fi

  while read -r files; do
    rm "$files"
    done < "$TmpStack"

  [ -f "$TmpStack" ] && rm $TmpStack
  print_info "Operazione completata!"

}

File di appoggio

Faccio uso di un file di appoggio (salvato nella directory di sistema /tmp) in cui vado a salvare il risultato della ricerca dei file che combaciano con l'estensione scelta dall'utente.

Precauzioni

Ad ogni iterazione verifico che:

  • Il file esista;
  • Il file non sia vuoto (è inutile effettuare operazioni successive se non ho corrispondenze di file);

Infine, elimino il file utilizzato.
