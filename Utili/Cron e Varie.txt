Aprire il file cron da terminale con questa riga di comando :

sudo crontab-e

Righe da inserire :

# Aggiornamenti ogni ora
0 * * * * sudo /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/Aggiornamenti.sh >> /home/kenobi/Documenti/aggiornamenti_log.txt 2>&1

# Pull e poi Push ogni 30 minuti
*/30 * * * * git -C /home/kenobi/Documenti/GitHub/CodicePersonale pull origin main >> /home/kenobi/Documenti/pull_log.txt 2>&1
*/30 * * * * git -C /home/kenobi/Documenti/GitHub/CodicePersonale push origin main >> /home/kenobi/Documenti/push_log.txt 2>&1

# Ottimizzazioni ogni 2 ore
0 */2 * * * sudo /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/Ottimizzazioni.sh >> /home/kenobi/Documenti/ottimizzazioni_log.txt 2>&1

#Se non inserisco source ~/.bashrc non carica gli alias e quindi non li riconosce


Per concedere i permessi aprire visudo :

sudo visudo

Aggiungere queste righe sotto alla riga # User privilege specification :

kenobi ALL=(ALL) NOPASSWD: /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/Aggiornamenti.sh
kenobi ALL=(ALL) NOPASSWD: /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/Ottimizzazioni.sh
kenobi ALL=(ALL) NOPASSWD: /bin/bash /home/kenobi/Documenti/GitHub/CodicePersonale/Ubuntu-Utility/SH-Personali/InstallApp.sh

Concedere permessi ed eseguibili :

sudo chmod u+x InstallApp.sh
sudo chmod u+x Aggiornamenti.sh
sudo chmod u+x Ottimizzazioni.sh
sudo chmod u+x PermessiSSh.sh

Permessi

# Modifica le autorizzazioni sul file chiave SSH privata
sudo chmod 600 ~/.ssh/id_rsa

# Aggiungi la chiave SSH privata all'agente SSH
ssh-add ~/.ssh/id_rsa

# Verifica che la chiave SSH privata sia stata aggiunta all'agente SSH
ssh-add -l
