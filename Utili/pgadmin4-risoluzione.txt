Guida per risolvere il problema di autenticazione con l'utente "postgres":

Passo 1: Disinstallazione di PostgreSQL
   ```bash
   sudo apt-get purge postgresql-15
   sudo apt-get purge postgresql-client-15
   sudo apt-get purge postgresql-client-common
   sudo apt-get purge postgresql-common
   sudo dpkg -l | grep postgres
   ```

Passo 2: Installazione di PostgreSQL
   ```bash
   sudo apt-get install postgresql-15 postgresql-client-15 postgresql-client-common postgresql-common
   ```

Passo 3: Cambiare la password di PostgreSQL
   ```bash
   sudo -u postgres psql
   ALTER USER postgres WITH PASSWORD 'nuova_password';
   \q
   ```

Passo 4: Verifica dell'autenticazione
   ```bash
   sudo -u postgres psql
   ```

Se la console si apre senza errori di autenticazione, l'autenticazione è stata effettuata correttamente.

Ora puoi creare nuovi database e utilizzare PostgreSQL normalmente.
